# Projet Symfony

Ce projet est construit avec **Symfony**, utilise **Composer** pour les dépendances PHP et **npm** pour la gestion des assets front-end.

---

## Prérequis

Assurez-vous d’avoir installé les outils suivants :

- PHP (version compatible avec Symfony)
- Composer
- Node.js (inclut npm)
- Git
- Un serveur local (Wamp, Xampp, Symfony CLI ou équivalent)

---

## Cloner le dépôt

```bash
git clone https://github.com/AkaneNigasaki/Symfony/
cd Symfony
```

## Installer les dépendence

```bash
composer install
npm install
```

## Lancement du server 
```bash
symfony server:start
```

**Assurer vous que votre Pare-feu autorise l'éxecution de git**
